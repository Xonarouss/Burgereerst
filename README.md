# BurgerEerst.nl

Next.js (App Router) + Tailwind + Supabase petition system with double opt-in and bilingual NL/EN pages.

## Quick start (local)
```bash
npm i
cp .env.example .env.local
npm run dev
```

## Required env vars
See `.env.example`.

## Supabase schema
See `supabase/schema.sql`.

## Deploy on Coolify
- Build command: `npm run build`
- Start command: `npm run start`
- Port: `3000`
- Add env vars from `.env.example`
